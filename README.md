# PressWatcher
在屏幕上显示按下的快捷键（Windows工具,可用于录屏）/Display the pressed shortcut key on the screen

---
在编辑视频时你是否烦恼需要用PR贴出来你的快捷键

在视频直播时你是否经常需要为同学们说明快捷键

来试试这款工具吧，他会在你按下快捷键时在屏幕上显示

目前只有简单功能，美观性也不是很好

正在开发，欢迎指教
